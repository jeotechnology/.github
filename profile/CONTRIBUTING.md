# Contributing to JEO Technology Profile

This project is owned by Joshua Edward Olds trading as JEO Technology.

> [!NOTE]
> Open an issue or confirm the intended change before starting substantial work. This keeps effort aligned and avoids wasted cycles.

## Before You Start

- Confirm scope with an issue or discussion before beginning large changes.
- Keep changes focused — one concern per pull request.
- Do not include secrets, private keys, credentials, tokens, or customer data in any commit.
- Preserve existing style and project conventions.

## Development Flow

1. Branch from `main` — use kebab-case branch names.
2. Make the smallest useful change.
3. Add or update tests and docs when behaviour changes.
4. Run validation before opening a pull request.

## Validation

Check markdown formatting and ensure all linked assets exist and are correctly referenced in `profile/repo-profile.json`.

By contributing, you agree that your contribution may be used under this project's license.
